# leetcode-problems

A clean, modern charcoal-themed progress tracker for standard DSA problem sets.

## Features

- **Replicated Segmented Gauge**: Displays progress matching the LeetCode circle design, divided into segments proportional to the distribution of Easy, Medium, and Hard problems.
- **LocalStorage Sync**: Saves progress (checkboxes) and starred items directly in your browser so you don't lose progress on page refresh.
- **Interactive Toolbar**: Supports searching by problem name, filtering completed or starred questions, shuffling a random unsolved question, and resetting all data.
- **Topic Categorization**: Easy navigation through 12 core DSA categories (from Arrays + Hashing to Dynamic Programming).
- **Direct Redirection**: Clicking a problem title redirects to the LeetCode description, and clicking the solution icon redirects directly to the LeetCode community solutions tab.

## Getting Started

To run this project locally, clone the workspace and install the dependencies:

```bash
bun install
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to view and use the tracker.
